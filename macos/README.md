# macOS 자동화 설정

현재 이 저장소에서는 macOS용 윈도우 매니저/바 인프라(yabai, skhd, sketchybar, borders)를 사용하지 않습니다. 관련 설정 파일은 제거되었습니다.

## 참고 메모
- 향후 다시 도입할 계획이 있다면 `README.md`에 적용 절차(백업 → 복사 → 서비스 재시작)와 필요한 권한 체크리스트를 새로 정리해주세요.
- 기존 macOS 환경에 다른 스크립트가 남아 있다면 `brew services list`로 활성화 상태를 확인하고, 불필요한 서비스는 `brew services stop <name>`으로 종료하는 것이 좋습니다.
