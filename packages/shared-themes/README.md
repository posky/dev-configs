# Shared Themes

## 목적

- 특정 도구 하나에 귀속하기 어려운 테마 문서와 공용 테마 후보를 관리합니다.

## 주요 파일

- `docs/everforest.md`: Everforest 색상 팔레트 참고 문서입니다.
- `docs/tokyo-night.md`: Tokyo Night 색상 팔레트 참고 문서입니다.
- `docs/lazygit/theme.yml`: Lazygit용 Tokyo Night Storm 테마 snippet입니다.

## 소유권 기준

- 특정 도구에서만 사용하는 테마 파일은 해당 도구 패키지의 `themes/`에 둡니다.
- 여러 도구에서 참고하거나 재사용할 수 있는 테마 문서와 설정 후보만 이 패키지에 둡니다.
